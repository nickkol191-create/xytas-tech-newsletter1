# XYTAS TECH — Newsletter

HTML email newsletter template for **XYTAS TECH ELECTRONICS** (security & telecom systems, Larissa, Greece).

- Single self-contained file: `xytastech-newsletter.html`
- Bulletproof, email-client-compatible (table layout, inline CSS, web-safe font fallbacks)
- Brand colors: navy `#101D2E` + blue `#29ABE2` / `#0066CC`
- Greek copy

## Use
Open `xytastech-newsletter.html` in a browser to preview, or paste its source into your email platform (Mailchimp, Brevo, etc.).

> Note: the logo is an inline-SVG recreation. For Gmail/Outlook, replace it with a hosted PNG (`<img src="...">`) — see the comment in the header.
